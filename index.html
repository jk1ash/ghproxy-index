GitHub Proxy Server


GitHub Proxy Server

GitHub 文件 , Releases , archive , gist , raw.githubusercontent.com 文件代理加速下载服务.

终端命令行

支持终端命令行 git clone , wget , curl 等工具下载.
支持 raw.githubusercontent.com , gist.github.com , gist.githubusercontent.com 文件下载.
注意：不支持 SSH Key 方式 git clone 下载.

git clone

git clone https://gh.jklash.top/ https://github.com/stilleshan/ServerStatus

git clone 私有仓库

Clone 私有仓库需要用户在 Personal access tokens 申请 Token 配合使用.
git clone https://user:your_token @gh.jklash.top/ https://github.com/your_name/your_private_repo

wget & curl

wget https://gh.jklash.top/ https://github.com/stilleshan/ServerStatus/archive/master.zip
wget https://gh.jklash.top/ https://raw.githubusercontent.com/stilleshan/ServerStatus/master/Dockerfile
curl -O https://gh.jklash.top/ https://github.com/stilleshan/ServerStatus/archive/master.zip
curl -O https://gh.jklash.top/ https://raw.githubusercontent.com/stilleshan/ServerStatus/master/Dockerfile

首页下载

在本页地址栏输入合规链接（参考以下链接）点击下载按钮
支持 raw.githubusercontent.com , gist.github.com , gist.githubusercontent.com 文件下载.
